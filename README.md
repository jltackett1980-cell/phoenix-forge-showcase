## What is in this repo

This is a curated sample of verified artifacts from Phoenix Forge — an autonomous genetic algorithm platform that generates chip designs, hardware schematics, and full-stack applications organized around human need.

### Silicon Handoff Packages
Three complete chip design packages with full EDA directory structure:
- **riscv_core** — RISC-V processor core with RTL, synthesis, constraints, and SPICE
- **alu_128bit** — 128-bit arithmetic logic unit with multiple SPICE netlists
- **fpu_ieee754** — IEEE 754 floating point unit

Each package contains: RTL source, synthesis output, timing constraints, SPICE simulation files, and datasheet.

### Hardware Design Samples

## Technical description

Phoenix Forge is a genetic algorithm evolver running autonomously on Android via Termux, federated with a Windows desktop through GitHub.

The coordination layer uses modular state components — not neural networks. The biological naming (amygdala, hypothalamus, prefrontal cortex) describes the functional role of each coordination module, not the implementation. They are state machines passing signals through a shared hormone bus. That is what they are and that is what they do.

The system evolves chip designs, hardware specifications, and software applications autonomously across 67 problem domains simultaneously.
